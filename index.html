Predicting Knee Osteoarthritis with Severity Grading

In this project, I worked with a Knee Osteoarthritis Dataset that included information on patients with varying degrees of knee osteoarthritis. My task was to develop a model that could accurately predict the severity of knee osteoarthritis in new patients based on their X-ray images. To achieve this, I utilized computer vision techniques, specifically transfer learning with MobileNetV2, and implemented the model in TensorFlow using Python. By leveraging these tools and techniques, I was able to create a robust model that achieved an accuracy of 74% in predicting the severity of knee osteoarthritis across 5 different types of severity. This is a significant achievement that provides valuable insights into the progression of knee osteoarthritis and highlights the potential of computer vision and machine learning in the medical field.

Semantic Segmentation for Remote Sensing

The Semantic Segmentation for Remote Sensing project involves using deep learning techniques to classify and segment remote sensing images into different land cover classes. Remote sensing images are collected from satellites, aircraft, and other platforms, and they provide critical information about the Earth's surface, including vegetation cover, urban areas, water bodies, and more..The project involves developing a deep learning model using TensorFlow to perform semantic segmentation on remote sensing data. The model achieved a high mean Intersection over Union (mIoU) score of 0.85 on the test set, surpassing some state-of-the-art methods in the field. The model accurately classified various land cover types, including water bodies, forests, urban areas, and agricultural fields, with an average IoU score of 0.8 across all classes.

Bike Rental Analysis

This project involved analyzing bike rental behavior in Washington D.C. between 2012 and 2018 using over 1 million records of raw data. The data was cleaned and prepared for analysis, resulting in a dataset of over 250k records. Various regression models were built and tuned to predict the number of bike rentals on a given day, with an achieved accuracy of over 82% using XgBoost with hyperparameter optimization using GridSearch. The project used data such as weather, location, and time to enhance the model's performance. The analysis aims to understand the factors that influence individuals' decisions to cycle and overall trends in bike rental behavior in the city.
